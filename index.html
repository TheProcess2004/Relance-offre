Guide d'utilisation
Maîtrisez FollowOffer en 5 minutes.
1
Configuration du compte
→ Configuration → Profil : renseignez votre nom, rôle, entreprise
→ Configuration → Email : saisissez votre adresse email expéditeur
→ Cliquez Sauvegarder — vos emails partent en votre nom via Brevo
2
Envoyer une offre
→ Cliquez + Envoyer une offre dans la sidebar
→ Remplissez : client, montant, objet, PDF (optionnel), contexte commercial
→ L'email part en votre nom — 3 relances programmées automatiquement (J+3, J+7, J+14)
3
Suivre les relances
→ Relances : offres en attente avec dates J+3 / J+7 / J+14
→ Calendrier : vue mensuelle de toutes les échéances
→ Cliquez 📧 Relancer manuellement ou activez les relances automatiques (Pro)
→ Marquez ✓ Gagnée ou ✕ Perdue — les relances s'annulent automatiquement
4
Tracker les ouvertures
→ Détection ouverture : voyez qui a ouvert, combien de fois, quand
→ Score 🔥 Très chaud = ouvert 3×+ · 🟢 Chaud = ouvert aujourd'hui
→ Appelez immédiatement un prospect chaud — il vient de lire votre offre
5
Analyser les performances
→ Revenus : CA gagné, taux de conversion, analyse des pertes
→ Rapport semaine : synthèse hebdomadaire exportable Excel
→ Fiches clients : historique complet, CA total généré par client
💡 Conseil
Ajoutez toujours un contexte commercial lors de l'envoi — hésitations client, budget, timing. Cela permettra de personnaliser chaque relance.
